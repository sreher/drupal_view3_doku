With the Description/Tag you can set the "View description" and the "View tag". The tag help you to organize your views. You only enter one tag. You can clone a view. When you click on the clone link on the upper right, all the option will copy to a new view, which you can give a new name. You can export a view. You can the copy and paste the code into a module or something else. It is a array with all option to a view.
